⚙️ Configuración del servidor ✓ Configurado
📌 Una sola configuración: al guardar aquí, tanto el panel como la app móvil (en el mismo navegador/equipo) quedan configurados automáticamente. Solo necesitas hacer esto una vez.
Pega aquí la URL que genera Google al implementar el script como app web.
Formato: https://script.google.com/macros/s/XXXX/exec
📋 Cómo configurar paso a paso
1. Crear el backend en Google Sheets:
  • Ve a drive.google.com → Nuevo → Hoja de cálculo de Google
  • En la hoja: Extensiones → Apps Script
  • Borra todo el código y pega el contenido de docs/google-apps-script.js
  • Guarda con Ctrl+S

2. Publicar como app web:
  • Clic en "Implementar" → "Nueva implementación"
  • Tipo: Aplicación web
  • Ejecutar como: Yo
  • Quién tiene acceso: Cualquier usuario
  • Clic en "Implementar" → autoriza los permisos
  • Copia la URL que aparece

3. Pegar la URL aquí arriba y clic en "Guardar y conectar"

4. ¡Listo! La hoja de cálculo se crea sola con el primer reporte enviado.
